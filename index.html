Bootcamp 2020: Solar System Viewer

My first programming project, which was completed during the 2 week Programming bootcamp at City University Of London (2020). This project was written in processing which is a simplier version of java. This processing project demonstrates my understanding of the use of classes and 3D graphics in processing. I created a solar system viewer in which your able to pick the desired planet from 1 - 9 (Same order as the solar system) which are stored in a class in another file. The class is being called in order to draw the 3D sphere and apply the texture onto it. Finally, your able to see a few facts about the planet.

Project 1: Programming In Java

This is my programming in java project, which was completed during year 1 term 2 at City University of London (2021). This project was written in Java using City 2d engine library in order to create game with animations, movement and collisions. This game is about destroying asteroids and prevnting the asteroids from hitting your or going past you; you'll need to destroy the asteroids in order to progress to the next level. Your able to save and load from different files. There's also background music which you can mute and also there's a play and pause button which your able to pause the game and then by pressing play your able to resume.
